# Rock-Paper-Scissors Game

This project is part of TOP curriculum.
It aloow the user to play the game with computer.

## Built With
- HTML
- Javascript

## Features
- User can select Rock, Paper or Scissors
- Computer randomly select its choice
- Display winner in each round
- Ends after 5 rounds
